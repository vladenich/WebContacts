Веб-сервис, предоставляющий API для доступа к списку контактов. 

Контакт представляет собой:
- номер телефона, 
- фамилию, 
- имя, 
- отчество. 

Реализованы следующие операции: 
- добавить новый контакт, 
- удалить контакт, 
- получить список всех контактов (с возможностью фильтрации по фамилии).
